# AI Sub-Agent Web

Simple example project that simulates an "AI sub-agent" which loads a plan and runs basic HTTP tests against a list of URLs.

## Features
- Load an agent plan from `agent_plan.md`
- Perform simple GET requests and record status codes and response times
- Produce a small analysis report

## Requirements
- Python 3.8+
- requests

Install requirements:
```bash
python3 -m pip install requests
```

## Usage
Run the test runner:
```bash
python3 runner.py
```

Edit `agent_plan.md` to change the agent plan or update `runner.py` to change the URLs under `urls_to_test`.

## Contributing
Open issues or submit PRs to improve tests, reporting, or add CI.
